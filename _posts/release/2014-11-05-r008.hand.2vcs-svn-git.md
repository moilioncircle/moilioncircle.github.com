# R008.练手.双版本控制:svn+git

折腾，穿越时空，离不开版本控制(VCS)。  
版本控制属配置管理(SCM)，是门大学问。  

svn简单上手快，适合高频提交，集中式在线协作。  
git强悍难驾驭，可离线可分布，但菜刀团队慎用。

## 主要话题

（1）poor man - version, backup   
（2）eclipse - local history  
（3）集中式: cvs，svn  
（4）分布式: hg，git  
（5）svn+git: 大胆折腾，自由反悔  
（6）互相添加 ignore
